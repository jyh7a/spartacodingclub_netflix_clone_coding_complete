# 0. 파이참 단축키
```
- 코드 모두 접기 / 펼치기 - ctrl + shift + - / ctrl + shift + +
- 찾기 / 바꾸기 - ctrl + f / ctrl + r 
- 파일 이름 변경 - shift + f6

- 참고
  - https://www.jetbrains.com/help/pycharm/finding-and-replacing-text-in-file.html
  - 구글
```
